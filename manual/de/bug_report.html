Wo wird der Fehlerbericht abgespeichert?

Der Fehlerbericht FreeDoko.bug_report.seed (wobei seed für die Spielnummer steht) wird im aufrufenden Verzeichnis abgespeichert. Unter Windows ist dies das FreeDoko-Verzeichnis, unter Linux auch das FreeDoko-Verzeichnis oder das Heimatverzeichnis.

Was steht alles in der Fehlerbericht?

In einem Fehlerbericht steht Ihre Beschreibung, die Version des Programms und der Spielverlauf. Sie können sich die Datei mit einem normalen Texteditor auch ansehen.

Wohin soll die Fehlerbericht geschickt werden?

Den Fehlerberricht schicken Sie bitte an freedoko@users.sourceforge.net. Sie können einen Fehler auch direkt auf der Sourceforge-Seite http://sourceforge.net/tracker/?func=add&group_id=38051&atid=421197 eintragen.
