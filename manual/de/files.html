Persönliche Dateien von FreeDoko

FreeDoko speichert einige persönliche Dateien ab. Diese finden sich unter Unix-Systemen im Unterverzeichnis .FreeDoko vom Heimatverzeichnis, unter Windows im Unterverzeichnis FreeDoko im Applikationsdaten-Verzeichnis des Benutzers.

Alle Dateien mit Einstellungen sind im normalen Textformat gespeichert. Das Format ist: Schlüsselwort = Wert. Leere Zeilen und Zeilen, die mit '#' beginnen, werden ignoriert.

FreeDokorc

Die Einstellungen werden in der Datei FreeDokorc gespeichert. Das Laden geschieht automatisch beim Programmstart, das Speichern automatisch, wenn Einstellungen geändert werden.

Version

In dieser Datei steht die Version von dem FreeDoko-Programm, welches Sie beim letzten Mal genutzt haben (Versionsnummer, ob Release, Erstellungszeit).

Genutzt wird die Datei um bei Versionswechsel ein Hinweisfenster mit den Neuerungen anzuzeigen, sowie automatisch Änderungen durchzuführen (natürlich auch mit einem Hinweis).

player.[1234]

In diesen vier Dateien werden die Spielerkonfigurationen (inklusive der KI) beim Starten eines Tuniers automatisch gespeichert Beim Starten von FreeDoko werden die Dateien geladen. player.1 ist der Spieler unten, player.2 der Spieler links, player.3 der Spieler oben und player.4 der letzte Spieler.

seed

In der seed-Datei werden die Startwerte der Turniere gespeichert. Damit haben Sie die Daten um frühere Spiele nochmal zu spielen.

Jeder Startwert wird in einer neuen Zeile an die Datei angehängt, gefolgt von dem Startspieler, dem Zeitpunkt und in der nächsten Zeile die Spielart (wenn sie nicht normal ist). Die Datei seed wird immer größer, maximal 75 Zeichen pro Spiel (d.h. frühestens nach 13.000 Spiele verbraucht die Datei ein Megabyte).

ai/

In diesem Verzeichnis werden die KI-Konfigurationen gespeichert. Dieses Verzeichnis wird ausgelesen, wenn eine KI-Konfiguration geladen werden soll.

rules/

In diesen Ordner werden die Regelsätze gespeichert. Dieses Verzeichnis wird ausgelesen, wenn ein Regelsatz geladen werden soll.

Der Regelsatz last ist der Regelsatz, der zuletzt genutzt wurde. Er wird beim Starten von FreeDoko automatisch geladen und beim Starten eines Tuniers automatisch gespeichert.

parties/

Im Verzeichnis parties werden die Partieen gespeichert. Die aktuelle Partie heißt current.
