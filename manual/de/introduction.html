Einführung in FreeDoko

Konfiguration

Menüpunkt Einstellungen → Einstellungen

  • falls die verwendete Sprache nicht deutsch ist: im Menü preferences den Punkt preferences anwählen, im Feld language deutsch auswählen
  • geben Sie unter Name ihren Spielernamen ein

Spielen

Turniereinstellungen
  • unter Regeln können Sie das Spiel weitestgehend ihren bevorzugten Regeln anpassen
  • drücken auf Turnier starten um mit dem Spiel zu beginnen
Vorbehalt
  • wählen Sie ggf. einen der verfügbaren Vorbehalte an oder entscheiden Sie sich unter Solo ein bestimmtes Solo zu spielen
  • klicken Sie anschließend auf ansagen
Spielverlauf
  • Klicken Sie auf die Karte, die Sie spielen möchten
  • An-/Absagen können Sie über den Menüpunkt Ansagen machen
Punkteübersicht
Der Menüpunkt Turnierpunkte im Menü Informationen zeigt ein Zusammenfassung der bisher gespielten Spiele an.

Viel Spaß beim Spielen.
