Turnier

neues Turnier

neues Turnier startet (ohne Nachfrage) ein neues Turnier.

Turnier laden

Bei Auswahl von Turnier laden können Sie mit Hilfe eines Dateiauswahldialogs eine Turnierdatei laden und das angegebene Turnier weiterspielen.

Turnier speichern

Immer am Ende eines Spiels können Sie das Turnier speichern. Automatisch wird das Turnier immer unter dem Namen current gespeichert.

Turnier beenden

Turnier beenden beendet das laufende Turnier und geht direkt zum Turnierendefenster. Ein noch laufendes Spiel wird nicht ausgwertet.

Programm beenden

Jederzeit können Sie FreeDoko beenden. Beim Beenden werden automatisch alle Einstellungen, Regeln und Spieler gespeichert.

Informationen

Während des Turniers können Sie verschiedene Informationen abrufen.

Kartenvorschlag

Immer wenn Sie eine Karte ausspielen müssen, können Sie sich vom Computer eine Karte vorschlagen lassen.

Letzter Stich

Jederzeit nach dem ersten Stich können Sie sich den letzten Stich / die letzten Stiche (siehe Regeln) ansehen.

Spielinformationen

Das Spielinformationsfenster gibt detaillierte Informationen über den bisherigen Spielverlauf. Da insbesondere die Teams der Spieler angezeigt werden ist dieser Menüpunkt in der veröffentlichten Version nicht vorhanden.

Regeln

Jederzeit während des Spielens können Sie sich die Regeln anschauen, in einem Turnier aber nicht ändern.

Turnierpunkte

Die Turnierpunkte umfassen die Punkte, die die Spieler in den einzelnen Spielen erreicht haben. Zu jedem vergangenen Spiel können Sie sich auch noch eine Spielzusammenfassung ansehen.

Spieler

Mit diesem Menüpunkt läßt sich die Konfiguration der Spieler ansehen.

Spielerstatistiken

Die Spielerstatistiken geben einen Überblick über die gewonnen Spiele und die genutzten Heuristiken.

Ansagen

Im Spiel können die Ansagen (Gegenansage, keine 120, keine 90, keine 60, keine 30, schwarz, Schweine, Hyperschweine) über dieses Menü getätigt werden. Wem das noch zu umständlich ist, der kann das Menü auch abreißen und so im separatem Fenster ansagen.

Einstellungen

Mit dem letzten Menüpunkt Einstellungen wird das Einstellungsfenster angezeigt und Sie können die Einstellungen ändern.
Die anderen Menüpunkte bieten einen Schnellzugriff auf die Einstellungen hebe zulässige Karten hervor, zeige alle Blätter, Kartensortierung.
Über den Menüpunkt Themen lässt sich ein Thema für Karten, Kartenrückseite, Symbole und Hintergrund auswählen.

Netzwerk

Das Netzwerk-Menü bietet Zugriff auf alles, was mit dem Netzwerkspiel in Verbindung steht.

Fehlermeldung

Mit einer Fehlermeldung können Sie das aktuelle Spiel speichern und uns zuschicken. Dieser Menüpunkt ist dafür da, daß Sie uns mitteilen können, wenn z.B. der Computer schlecht spielt, so daß wir das Programm verbessern können.

Hilfe

Hilfeübersicht

Die Hilfeübersicht wird im angegebenen HTML-Betrachter angezeigt. Wird die Hilfe nicht auf dem Computer gefunden wird die Dokumentation im Internet geladen.

Lizenz

FreeDoko steht unter der GNU General Public License, eine Kurzform können Sie sich unter diesem Menüpunkt anschauen.

Weiterhin können Sie sich die Lizenzen für den Kartensatz und gegebenenfalls die Symbole anschauen.

Änderungsliste

Die Änderungsliste enthält die Änderungen seit der Version 0.6.3.

Über

Das Fenster Über enthält u.a. die Autoren, die Adresse der Internetseite und die E-Mail-Adresse zu FreeDoko.

erzeuge Fehler

Mit diesem Menüpunkt kann ein Fehler erzeugt werden, der wie ein interner Fehler behandelt wird. Insbesondere kann so ein Speicherabzug erstellt werden.
