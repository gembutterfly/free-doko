Einstellungen

Einstellungen können unter dem Menupunkt „Einstellungen:Einstellungen” geändert werden. Sie werden automatisch beim Programmstart geladen und beim Verlassen gespeichert. Eine Änderung tritt sofort in Kraft, es ist keine Bestätigung erforderlich (vgl. GNOME Kontrollzentrum).

Gespeichert werden die Einstellungen in der Datei FreeDokorc im privaten FreeDoko-Verzeichnis (Linux: ~/.FreeDoko, Windows: FreeDoko im Verzeichnis für die persönlichen Programmeinstellungen). Die Syntax ist dabei: Kennwort: Wert.

Im folgenden werden die Einstellungen im einzelnen erklärt:

Allgemein

In dieser Rubrik werden allgemeine Einstellungen wie der Name und die Sprache getätigt. Die Zeiten werden in Sekunden eingestellt mit einer Genauigkeit von Zehntelsekunden (das sollte reichen).

Name
Der Name vom Spieler. Ist kein Name angegeben, wird der Name vom System übernommen.
Sprache
Die Sprache für die Texte. Aus den installierten Sprachen kann eine aus der DropDown-Liste ausgewählt werden. Standard übernimmt die Sprache vom System.
HTML-Betrachter
Welcher Browser für die Anzeige von den Hilfe-Seiten genutzt werden soll. Ist keiner angegeben, so wird unter Windows in der Registry gesuch, unter Linux Browser von der internen Liste (firefox, epiphany, galeon, mozilla, opera, konqueror, netscape, w3m, links, lynx) getestet.
Sound
Diese Einstellung gibt an, ob Sound genutzt werden soll.
zeige Fehlermeldung-Knopf im „Spielende”-Fenster
Wenn diese Einstellung aktiviert ist, kann direkt vom Spielende-Fenster ein Fehlerbericht erstellt werden.
Speicher die Fehlermeldungen auf dem Desktop
Wenn diese Einstellung aktiviert ist, werden die Fehlermeldungen auf dem Desktop (im Ordner 'FreeDoko.BugReports') gespeichert. Ansonsten werden sie im Verzeichnis mit den privaten Daten von FreeDoko (s.o.) gespeichert.
Turnieränderungen speichern
Normalerweise werden Turnieränderungen, das sind Regeln und die Spieler, automatisch gespeichert. Mit dieser Einstellung wird das Speicherung unterdrückt. Wenn ein Fehlerbericht nachgespielt wird, wird diese Einstellung automatisch auf aus gestellt.
Zusätzliche Turnierneistellungen
Zeigt die Turniereinstellungen „Startwert“ und „Ausspieler“ mit an. Die Einstellung wird benötigt, um Turniere nachspielen zu können. Ist sie abgewählt, wird der Startwert zufällig gewählt und der Ausspieler ist der erste Spieler.
HTML-Betrachter
Das Programm, mit dem HTML-Dateien (die Hilfe und das Handbuch) geöffnet werden.

Verhalten

Verzögerung beim Kartenausspielen
Zwischen dem Ausspielen von zwei Karten vergeht (mindestens) die angegebene Zeit.

Unterstützung

zeige Solospieler im Spiel
Wenn ein Spieler ein Solo, eine Hochezeit, eine Armut oder einen Genscher spielt wird er mit der Spielart markiert.
Schweine automatisch ansagen
Die Schweine und Hyperschweine werden automatisch angesagt und zwar zum letztmöglichen Zeitpunkt. Sie können die Ansage auch mit dieser Einstellung auch vorher tätigen.
zeige ob zulässig
Es wird durch den Mauszeiger gezeigt, welche Karten im laufenden Stich zulässig sind.
hebe zulässige Karten hervor
Alle unzulässigen Karten werden grau dargestellt.

Hilfe

zeige alle Blätter
Wenn es die Regel zuläßt, werden die Blätter aller Spieler und bei der Armut die geschobenen Karten gezeigt.
zeige Blätter (KI-Information)
Die Blätter der Mitspieler werden nach den Informationen der KI angezeigt. Helle Karte kann der Mitspieler haben, normal dargestellte muß er besitzen.
Warnung: Da vor allem am Spielanfang viele Karten dargestellt werden verlangsamt die Einstellung das Programm. Außerdem sind die Blätter nicht wirklich übersichtlich.
zeige bekannte Teams im Spiel
Sobald für einen Spieler die Teamzugehörigkeit bekannt wird (z.B. durch Ansagen und Spielen der Kreuz Dame) wird dessen Teamzugehörigkeit angezeigt.
zeige Teams (KI-Information)
Bei den Mitspielern wird durch hell dargestellte Symbole das Team angezeigt, wie die KI es sich denkt (sofern es nicht bekannt ist).
zeige Punkte der Stichhaufen
Auf den Stichhaufen steht, wie viele Punkte der Spieler schon gesammelt hat.
automatischer Kartenvorschlag
Immer wenn der menschliche Spieler dran ist wird automatisch eine Karte vorgeschlagen, diese wird in Richtung Tischmitte geschoben.

Informationsfenster

Für die verschiedenen Spielinformationen können Fenster aktiviert werden, die sich bei Bedarf auch automatisch wieder schließen. Geschlossen werden können die Fenster durch den entsprechenden Knopf oder einfach über einen Klick in die Spielfläche.

Alle Informationen sind auch über die Symbole sichtbar, wenn man sich nur auf die Symbole verläßt kann es aber leichter passieren, daß man etwas übersieht. Insbesondere beim Genscher kann sich das sehr nachteilig auswirken.

Spieltyp

zeige Fenster „Spieltyp”
Beim Start eines Spiels wird der Spieltyp angezeigt, wenn kein normales Spiel vorliegt.
Wenn die Einstellung deaktiviert ist und Schweine automatisch ansagen deaktiviert ist und Schweine beim Spielstart angesagt werden müssen, können bei einem Farbsolo keine Schweine angesagt werden.
schließ Fenster „Spieltyp” automatisch
Das Fenster „Spieltyp” wird automatisch geschlossen.
Verzögerung beim Schließen vom „Spieltyp”-Fenster
Nach der angegebenen Zeit wird das „Spieltyp”-Fenster automatisch geschlossen.

Voller Stich

zeige Fenster „Voller Stich”
Das Fenster „Ganzer Stich” zeigt bei jedem vollen Stich an, weclher Spieler den Stich erhält und welche Sonderpunkte durch den Stich erreicht werden.
zeige Fenster „Voller Stich” bei Sonderpunkten
Wenn das Fenster „Voller Stich” normalerweise nicht angezeigt werden soll kann die Anzeige bei Vorliegen von Sonderpunkten aktiviert werden.
schließ vollen Stich automatisch
Der volle Stich wird automatisch geschlossen (ansonsten nur mit einem expliziten Mausklick).
Verzögerung beim Schließen vom vollen Stich
Nach der angegebenen Zeit wird der volle Stich automatisch geschlossen und der nächste Stich geöffnet. Durch einen Mausklick kann der Stich auch vorher geschlossen werden.

Hochzeit

zeige Fenster „Hochzeit”
Bei der Hochzeitsfindung wird ein Fenster angezeigt, der das Paar angibt.
schließ Fenster „Hochzeit” automatisch
Das Fenster „Hochzeit” wird automatisch geschlossen.
Verzögerung beim Schließen vom „Hochzeit”-Fenster
Nach der angegebenen Zeit wird das „Hochzeit”-Fenster automatisch geschlossen.

Genscher

zeige Fenster „Genscher”
Bei einem Genscher wird ein Fenster angezeigt, das das neue Re-Paar anzeigt.
schließ Fenster „Genscher” automatisch
Das Fenster „Genscher” wird automatisch geschlossen.
Verzögerung beim Schließen vom „Genscher”-Fenster
Nach der angegebenen Zeit wird das „Genscher”-Fenster automatisch geschlossen.

Ansage

zeige Fenster „Ansage”
Wird eine Ansage getätigt, wird ein Fenster angezeigt. In diesem Fenster kann eine Gegenansage getätigt werden, ansonsten nur über das Menü.
schließ Fenster „Ansage” automatisch
Das Fenster „Ansage” wird automatisch geschlossen.
Verzögerung beim Schließen vom „Ansage”-Fenster
Nach der angegebenen Zeit wird das „Ansage”-Fenster automatisch geschlossen.

Schweine

zeige Fenster „Schweine”
Wenn ein Spieler Schweine oder Hyperschweine ansagt wird ein Fenster angezeigt.
Wenn die Einstellung deaktiviert ist und Schweine automatisch ansagen deaktiviert ist und Hyperschweine beim Spielstart angesagt werden müssen, können bei einem Farbsolo keine Hyperschweine angesagt werden.
schließ Fenster „Schweine” automatisch
Das Fenster „Schweine” wird automatisch geschlossen.
Verzögerung beim Schließen vom „Schweine”-Fenster
Nach der angegebenen Zeit wird das „Schweine”-Fenster automatisch geschlossen.

Netzwerk

Verbindungs-Hinweis
Gibt einen Hinweis, daß sich die Netzwerk-Verbindungen geändert haben.
Chat-Hinweis
Gibt einen Hinweis, daß eine neue Chat-Nachricht angekommen ist.

Kartensortierung

Die Karten koennen geordnet oder gemischt dargestellt werden. Bei der Ordnung wird nach den fünf Spielfarben Trumpf, Kreuz, Pik, Herz und Karo sortiert, bei jeder Farbe kann noch zwischen auf- und absteigenden Wert gewählt werden.

Erscheinungsbild

Tisch

eigenes Blatt unten
Wenn aktiviert, wird das Blatt des menschlichen Spielers am unteren Tischrand angezeigt.
Tischrotation
Gibt an, an welcher Position der erste Spieler sitzt (es wird im Uhrzeigersinn gedreht).
rotiere Stichkarten
Wenn die Stichkarten rotiert werden, werden die Karten so dargestellt, als ob jeder Spieler seine Karte gerade in die Mitte geschoben hat. Werden sie nicht rotiert, so liegen die Stichkarten aufrecht.

Karten

normale Kartengröße
Die Karten werden in der in dem Kartensatz vorliegenden Größe dargestellt.
Kartenhöhe
Die Karten werden skaliert, so daß die Höhe die angegebene erreicht. Das Verhältnis von Breite und Höhe bleibt erhalten.

Thema

Hintergrund
Mit dem Hintergrund kann angegeben werden, welche Graphik als Tischhintergrund (gekachelt) genutzt wird.
Kartenspiel
Das Kartenspiel beinhaltet die Graphiken für die Karten und die Symbole.
Kartenrueckseite
Die Kartenrückseite kann von den Rückseiten vom gewählten Kartenspiel ausgewählt werde.

Schriftarten, Farben

Schriftart der Namen
Hier kann die Schriftart für die Namen der Spieler im Spielfeld angegeben werden.
Farbe der Namen
Hier kann die Farbe für die Namen der Spieler im Spielfeld angegeben werden.
Farbe des Namens des aktiven Spielers
Dem aktiven Spieler kann eine andere Farbe gegeben werden, so daß sich dieser leichter erkennen läßt. Der aktive Spieler im Spiel ist der, der die nächste Karte ausspielen muß (d.h. beim vollen Stich auch der Gewinner des Stiches), wenn das Spiel beendet ist werden die Gewinner angezeigt, beim Turnierende der/die Turniergewinner.
Schriftart der Punkte der Stichhaufen
Diese Schriftart wird für die Punkte der Stichhaufen verwendet.
Farbe der Punkte der Stichhaufen
Die Farbe der Schrift sollte an den Kartenhintergrund angepaßt werden.
Farbe des Schiebepfeils
Diese Farbe wird für den Schiebepfeil verwendet.

Splash-Screen

zeige Splash-Screen
Wem der Splash-Screen stört, der kann ihn hier abschalten.
transparenter Splash-Screen
Der transparente Splash-Screen sieht meiner Meinung nach besser aus — aber man hat ja die Wahl.

Sonstige

Automatisches Speichern
Es werden die folgenden Daten automatisch gespeichert:
  • Version
  • Einstellungen
  • Regeln
  • Spieler
  • Tunier
Diese Einstellung kann über die Kommandozeilenoption --no-automatic-savings abgestellt werden.
