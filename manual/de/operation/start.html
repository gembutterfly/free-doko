Programmstart

Normalerweise startet FreeDoko dieses Programm. Unter Windows ist für den Start die Batch-Datei FreeDoko.bat vorgesehen, die sorgt dafür, daß die dll-Dateien von gtkmm eingebunden werden können.
Kommandozeilenargumente können mit FreeDoko -? eingesehen werden.

Beim ersten Start von FreeDoko wird ein Begrüßungsfenster angezeigt. Beim Wechsel zu einer neuen Version werden die Änderungen aufgelistet. Wenn Verzeichnisse erstellt werden (normalerweise nur beim ersten Start) werden diese auch angegeben. Diese Verzeichnisse enthalten die persönlichen Daten.

Das Startfenster kann in den Einstellungen geändert und abgeschaltet werden.

Startfenster

Wenn irgendwelche Dateien (Graphiken) nicht geladen werden können gibt es einen Hinweis, welche Datei fehlt, anschließend beendet sich FreeDoko.

Wenn FreeDoko und die zugehörigen Daten geladen sind wird das Hauptfenster und das Turniereinstellungen-Fenster angezeigt.
